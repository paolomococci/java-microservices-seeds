# Example of use

## How to built:

```
$ sudo docker build --tag mongodb_image .
$ sudo docker run -d --name mongodbstore -e MONGO_INITDB_ROOT_USERNAME=dbadmin -e MONGO_INITDB_ROOT_PASSWORD=password -p 27017:27017 mongodb_image
$ sudo docker images
$ sudo docker ps --all
```
